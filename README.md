Source code of the manuscript "Quantifying risk modifiers of
hereditary hemochromatosis using genomic and electronic health record
data from FinnGen and UK Biobank"

# Description of files

## src directory

R source code files.

File | Description
-----|------------
common.R | Common R utility functions
create_article_figures.qmd | R document that creates final figures and tables 
finngen_explore.qmd | Exploration for FinnGen data
finngen_model.qmd | Modeling hemochromatosis in FinnGen
finngen_hc_progression.qmd | Modeling hemochromatosis severity in FinnGen
ukbb_explore.qmd | Exploration for UKBB data
ukbb_model.qmd | Modeling hemochromatosis in UKBB
interaction_with_mvmapit.qmd | Interaction GWAS with mvmapit in FinnGen
regenie_interactions.qmd | Interaction GWAS with Regenie in FinnGen
hla_hemochromatosis_gwas_R12.qmd | HLA hemochromatosis GWAS in FinnGen
